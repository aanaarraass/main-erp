OpenEduCat BigBlueButton Integration

Manage Session with BigBlueButton

Online Demo

This module allows you to manage and create calender online meeting using session on BigBlueButton.

Contact Us

Now, Go to menu Timetables -> Sessions. Here Select Particular Session on which you want to create meeting.

Click on button online meeting the calender will open with all session details

Click on button Create Bigbluebutton meeting. the wizard will open .In that wizard, auto start recording, dial number and record meeting can be set from settings of openeducat. Here you have to add attendee password and moderator password. And by clicking on Invite via Email boolean, the email will be sent to all the attendees of the session for this video conference meeting

Here under the Invitation Tab, you can see the url and attendee password of all the attendees of that session.

By clicking on the url ,meeting will open in a new tab/window.

The attendees will receive a link to join with the invitation mail.
